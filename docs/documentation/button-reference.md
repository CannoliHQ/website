# Button Reference

!!! tip
    This reference uses the default **Plumber** button labels. If you've changed your [Button Labels](settings.md#input) to **Redmond** or **Shapes**, confirm and back are swapped: confirm is the south button and back is the east button.

## Platform List

| Button | Action                              |
|--------|-------------------------------------|
| X      | Open settings                       |
| Y      | Open Nonna's Kitchen (when running) |
| Start  | Open context menu                   |
| Select | Toggle reorder mode                 |

### Reorder Mode

| Button          | Action             |
|-----------------|--------------------|
| D-Pad Up / Down | Move selected item |
| Start           | Confirm reorder    |

## Games List

!!! tip
    These controls apply to all three kinds of games list; platform, favorites and collections.

| Button | Action                                             |
|--------|----------------------------------------------------|
| A      | Launch game or open folder                         |
| X      | Resume game, if save state exists                  |
| L1     | Previous platform, [if enabled](settings.md#input) |
| R1     | Next platform, [if enabled](settings.md#input)     |
| Start  | Open context menu                                  |
| Select | Toggle multi-select                                |

### Multi-Select Mode

| Button          | Action                 |
|-----------------|------------------------|
| D-Pad Up / Down | Move selection         |
| A               | Toggle item checkbox   |
| Start           | Confirm selection      |
| Select          | Exit multi-select mode |

## On-Screen Keyboard

| Button        | Action                          |
|---------------|---------------------------------|
| D-Pad         | Move keyboard cursor            |
| A             | Input character                 |
| B             | Backspace                       |
| X             | Insert space                    |
| Y             | Cancel input                    |
| L1 / R1       | Move text cursor left / right   |
| L2 / R2       | Move text cursor to start / end |
| Start         | Confirm input                   |
| Select (Tap)  | Toggle capslock                 |
| Select (Hold) | Toggle symbols                  |

## In-Game

| Button          | Action                          |
|-----------------|----------------------------------|
| Menu            | Open / close In-Game Menu        |
| D-Pad Up / Down | Navigate menu options            |
| A               | Select option                    |
| B               | Back / close menu                |

### Save / Load State

| Button      | Action                                  |
|-------------|-----------------------------------------|
| Left / Right | Cycle between slots                    |
| A           | Save or load the selected slot          |
| Y           | Delete selected slot                    |
| X           | Undo last save / load (one-minute window) |

### Switch Disc

| Button       | Action              |
|--------------|----------------------|
| Left / Right | Pick disc            |
| A            | Confirm disc switch  |

## Color Picker

| Button | Action             |
|--------|--------------------|
| D-Pad  | Move in color grid |
| A      | Select color       |
| B      | Cancel             |
| X      | Open hex input     |
